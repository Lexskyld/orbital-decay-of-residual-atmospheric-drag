# orbital-decay-of-residual-atmospheric-drag
This project simulates the orbital decay of a satellite in low Earth orbit by residual atmospheric drag, used a layered model to calculate the density changes between the thermosphere and altitudes.
Findings: by increasing the satellite's surface area from 0.01 m^2 to 0.1 m^2, the lifetime of orbit reduced from 1412.002315 days to 141.0069444 days, therefore demonstrated the importance of drag sails in reducing space debris.
